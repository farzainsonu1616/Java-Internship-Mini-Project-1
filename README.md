# Online Quiz System

## Overview
The Online Quiz System is a Java-based desktop application that allows users to log in and attempt timed quizzes. 
The system loads questions from a MySQL database, evaluates answers, and displays the final score.

## Features
- User Login Authentication
- Random Question Loading
- Timer-based Quiz
- Automatic Scoring
- Clean Swing User Interface
- MySQL Database Integration

## Technologies Used
- Java (Core Java)
- Swing (UI)
- MySQL Database
- JDBC
- IntelliJ IDEA

## How to Run
1. Start MySQL service.
2. Open the project in IntelliJ IDEA.
3. Run Main.java.
4. Login using valid credentials.
5. Attempt the quiz.
